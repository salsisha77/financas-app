# 💰 Finanças App

Aplicativo web simples para controle financeiro pessoal diretamente no navegador.

👉 Projeto criado para praticar JavaScript e desenvolvimento web front-end.

🌐 **Acesse o app online:**  
https://salsisha77.github.io/financas-app/

---

## ✨ Funcionalidades

• Adicionar receitas e despesas  
• Cálculo automático do saldo  
• Interface simples e rápida  
• Dados salvos no navegador (LocalStorage)  
• Funciona direto no navegador, sem instalar nada  

---

## 🚀 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage  

---

## 📱 Como usar

1. Abra o site do projeto
2. Adicione seus ganhos e gastos
3. Veja o saldo atualizado automaticamente

Não precisa criar conta ou instalar nada.

---

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

- Manipulação do DOM  
- Lógica com JavaScript  
- Armazenamento local no navegador  
- Criação de aplicações web simples
